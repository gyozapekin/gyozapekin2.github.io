【ギャラリー用】
--------------------------------------------------

gallery/ サブフォルダを作成して以下を配置してください：

  思い出の壁：
    memory_01.jpg ～ memory_06.jpg（以降も追加可）

  各期の写真：
    2022q1_01.jpg, 2022q1_02.jpg, 2022q1_03.jpg
    2022q2_01.jpg, 2022q2_02.jpg, 2022q2_03.jpg
    …
    2024q1_01.jpg, 2024q1_02.jpg, 2024q1_03.jpg

  ※ 写真が多い場合は 01, 02, 03, 04... と増やしてください
  ※ gallery.html の <div class="gallery-grid"> 内に
     <img src="images/gallery/ファイル名.jpg"> を追加するだけで増やせます

--------------------------------------------------
【画像の探し方（ジンドゥー保存ファイルから）】
--------------------------------------------------

保存したHTML（例: おいしい餃子の焼き方 - gyoza-pekin ページ！.html）と
一緒に保存されている「_files」フォルダの中に画像が入っています。

例：
  おいしい餃子の焼き方 - gyoza-pekin ページ！_files/
    ├── image001.jpg  ← これが目的の画像
    ├── image002.jpg
    └── ...

ブラウザで保存したHTMLを開き、どの画像が何かを確認しながら
このimagesフォルダにコピーしてください。
