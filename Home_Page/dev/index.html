北京のこだわり

海鮮餃子 北京では、毎日手作りにこだわっています。新鮮な海老、たこ、貝柱を使った海鮮餃子は当店の看板メニュー。皮から手作りし、一つ一つ丁寧に包んでいます。

テレビ番組「なるみ・岡村の過ぎるTV」でも紹介された当店の味を、ぜひ一度お試しください。

メニューを見る → / ブログ・メディア情報 →

今月・来月の休み

毎週火曜日・水曜日が定休日です。臨は赤字、臨時営業は緑字で表示されます。

いらっしゃいませ

海鮮餃子の北京は 大阪府枚方市 宮之阪にある中華料理店です。
新鮮な豚肉と減農薬野菜を使った手作りの海鮮餃子をお楽しみいただけます。
もちろん餃子以外の一品料理もおいしいものづくしです。
あげそば、酢豚、唐揚げ、天ぷら、焼飯など、その他多数のメニューをご用意しておりますので、
是非おなかをすかせてお越しください。
